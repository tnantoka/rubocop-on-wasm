# RuboCop on Wasm

Run RuboCop on a web browser with [ruby.wasm](https://github.com/ruby/ruby.wasm).

http://rubocop-on-wasm.tnantoka.com/

![](/docs/screenshot.png)

## Developmennt

```bash
$ cd rubocop.wasm
$ rake

$ npm i
$ npm run dev
```
